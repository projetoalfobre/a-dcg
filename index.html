Welcome!

A-DCG (Alfobre Dublin Core Generator) is a web tool project developed to support the creation of Dublin Core (DC) metadata. The tool comes in two flavors: the simple version only uses the 15 DC main elements and the advanced version generates Dublin Core metadata code using the more detailed qualified elements and encoding schemes.

This tool can be used online (like in this website) and offline. If you want to use it offline, just download de source code from GitHub and use a browser to open the index.html file. If you want, you can also install this tool in your own server or create your own version.
